# 🍰 Cardápio de Doces

Um projeto de cardápio digital desenvolvido com HTML, CSS e JavaScript, focado em um design moderno, responsivo e agradável para apresentar produtos de uma confeitaria ou doceria.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- Exibição dos doces em um layout organizado
- Interface moderna e intuitiva
- Design responsivo para diferentes dispositivos
- Navegação simples e agradável

## 📂 Como executar

1. Clone este repositório:

```bash
git clone https://github.com/Marcosdeivide/cardapio-de-doces.git
```

2. Abra o arquivo `index.html` no navegador.

## 🎯 Objetivo

Este projeto foi desenvolvido para praticar conceitos de desenvolvimento Front-End, incluindo:

- Estruturação com HTML semântico
- Estilização com CSS
- Responsividade
- Organização de arquivos
- Boas práticas de desenvolvimento
